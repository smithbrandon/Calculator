Keyboard Shortcuts

The calculator can be used by clicking on the buttons with the mouse or using the keyboard. In order to use the keyboard the body of the page must be clicked first. By clicking on this message to continue, the keyboard is enabled.

Please add comments if you find a functionality isn't working or have something you feel should be added. Thank you for viewing.

Element Shortcut
(.) Numberic Keypad or "." Key
(=) Numberic Keypad or '=' Key or Return
(×) Numberic Keypad or '*' Key
(+) Numberic Keypad or '+' Key
(-) Numberic Keypad or '-' Key
(÷) Numberic Keypad or '/' Key
(×) '%' Key
(√) '^' Key
(AC) 'Delete' Key or 'ESC' Key

*Note - backspace has been disabled as to not conflict with webpage nav.

Click anywhere to continue press "i" to return to shortcuts
